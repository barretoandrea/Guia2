# Registro de Productos

## Descripción
Aplicación web desarrollada con HTML, CSS y JavaScript que permite registrar productos mediante un formulario dinámico.

## Funcionalidades
- Registro de productos.
- Validación de nombre, precio y stock.
- Mensajes de error y éxito.
- Agregado dinámico de productos en una tabla.
- Diseño responsive.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Cómo ejecutar el proyecto
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo `index.html` en un navegador.
